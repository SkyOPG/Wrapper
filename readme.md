# SkyWrapper
- A raylib wrapper for SkyOPG's apps and games
# Usage
- Clone this repo
```sh
git clone https://github.com/SkyOPG/Wrapper
```
- For windows users, download mingw64, linux users can chill with gcc and make
## windows building
```sh
make win
```
## linux building
```sh
make linux
```
- for multiplatform
```sh
make global
```
